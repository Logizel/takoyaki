Takoyaki

GitHub Activity Pinger for Discord — get real-time push, PR, and issue notifications, plus contribution stats and streak heatmaps.

Add to Discord

Free • Open source • Powered by GitHub & Discord

⚡ Features

🔔

Real-time Pings

Get instant Discord notifications when you push, open a PR, create an issue, or start a new repo. Private repos stay private.

📊

Contribution Stats

View your GitHub contribution count, daily averages, and compare with others using /stats commands — powered by GitHub's GraphQL API.

🔥

Streak Heatmap

Visualize your contribution history with an ANSI heatmap grid showing 6 months of activity — right inside Discord.

🏆

Leaderboards

Compete with your server members using daily and yearly contribution leaderboards. See who's been most active.

⌨️ Commands

/stats me Your contribution count and daily average
/stats compare Compare with another user
/stats top Yearly contribution leaderboard
/stats top-day Today's contribution leaderboard
/stats streak View your 6-month heatmap
/github link Link your GitHub account
/github unlink Unlink your GitHub account
/setchannel Set the notification channel

🚀 Get Started

1

Install the GitHub App

Click Add to Discord above and authorize the GitHub App. You'll be redirected back to Discord.

2

Link your account

Run /github link in any channel where Takoyaki is present. Follow the OAuth flow to connect your GitHub account.

3

Set a notification channel

Run /setchannel in the channel you want GitHub notifications delivered to. That's it — you're all set.

🔒 Privacy

Takoyaki stores your GitHub username and an OAuth access token needed to fetch your contribution stats. Your access token is never shared, logged, or exposed. It is stored in our database and used solely for GitHub API requests you initiate via /stats commands. Push, PR, and issue events are processed in real-time and not stored beyond commit counts for streak tracking.
